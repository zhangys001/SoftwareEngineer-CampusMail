# SoftwareEngineer-CampusMail

校园快递收发系统 — 软件工程课程设计（第9组）

## 项目简介

基于 Flask + SQLite 的校园快递收发管理系统，支持学生、驿站工作人员、管理员三种角色。

### 功能模块

**学生端 (H5)**
- 快递查询：通过单号或取件码查询快递状态
- 我的快递：查看待取/已取快递列表
- 扫码取件：出示取件码供工作人员核销
- 预约寄件：在线填写寄件信息并提交预约
- 代取授权：授权其他同学代为取件

**员工端**
- 工作台：今日入库/出库统计、滞留件预警
- 快递入库：扫描单号、分配货架、生成取件码
- 出库核销：扫描取件码完成出库
- 货架管理：查看货架状态和库位使用率

**管理员端**
- 数据概览：运营指标、趋势图、公司分布
- 用户管理：增删改查用户账号
- 公告管理：发布/撤回系统公告
- 快递统计：按日期范围查询并导出报表

## 技术栈

- 后端：Python 3 + Flask + SQLAlchemy
- 数据库：SQLite
- 前端：HTML + CSS + Jinja2 模板

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务（自动初始化数据库和示例数据）
python run.py
```

访问 http://localhost:5000
